# Sim Platform Brand Standards (MVP)

## 1) Brand personality
- **Confident, clear, academic-modern**
- Avoid overly playful UI; keep trust and legibility high
- Add color through accents and gradients, not noisy surfaces

## 2) Color system

### Core palette
- **Primary / Indigo**: `#4F46E5`
- **Primary dark**: `#3730A3`
- **Accent / Cyan**: `#06B6D4`
- **Accent / Violet**: `#8B5CF6`
- **Success**: `#16A34A`
- **Warning**: `#D97706`
- **Danger**: `#DC2626`

### Neutrals
- Background: `#F8FAFC`
- Card surface: `#FFFFFF`
- Text primary: `#0F172A`
- Text secondary: `#475569`
- Border: `#E2E8F0`

## 3) Typography
- Font: Geist Sans (already configured)
- H1: bold, high contrast
- Body: 16px equivalent, line-height ~1.5
- Secondary text should pass contrast standards

## 4) UI primitives
- **Cards**: soft border + subtle shadow + rounded-xl
- **Buttons**:
  - Primary: indigo gradient + white text
  - Secondary: white with border
  - Ghost: low-emphasis text action
- **Badges**: use semantic colors for statuses

## 5) Layout
- Max width for content pages: `max-w-6xl`
- Generous whitespace (24–32px around sections)
- Clear section hierarchy with headings + helper text

## 6) Accessibility
- Color contrast WCAG AA baseline
- Keyboard focus visible on all interactive elements
- Don’t rely on color alone for status: include text labels

## 7) Tone in visuals
- “Calm, capable, modern classroom software”
- Prefer clean gradients and structured cards over playful illustrations
